# Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Dark/Light theme toggle
- Animated sections with AOS
- Downloadable resume
- Social media links

## Usage

Open `index.html` in your browser.

## Customization

- Update your profile image in `Img/`
- Edit your details in `about.html`
- Add your projects in `projects.html`
- Update social links in `footer.html`
