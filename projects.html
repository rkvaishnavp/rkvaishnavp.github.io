Hand Written Digit Recongnition in FPGA

Verilog, Digital Design, Xilinx Vivado, Artix A7

Trained a Neural Network with perceptron neuron in Python for Handwritten digit recognition(used MNIST Dataset). Currently Working on Accelerating the digit recongition neural network inference in FPGA. The Weights and Biases data of the neurons is sent to FPGA using UART communication which is then stored in BlockRam, then image data is sent again through the UART Which is then passed to the accelerator module in FPGA which does parallel computing to recognise the digit. The recognised number is sent back to the Computer again in UART

Display and VGA Core

Verilog, Python, Digital Design, Xilinx Vivado, Artix A7

Created a Display Core in verilog which generates VGA signals (VGA Standards (red, green, blue, hsync, vsync) in 4 bit format RGB444. The core takes mem file as image source in binary uncompressed format(This was generated using Image Module python package which takes a cropped image (640x480) as input). To test if the core is generating the signal according to the standards, wrote a simple decode unit in verilog HDL which takes VGA signals and dumps the data to the log file. Using python scripts the input image and output data was compared to check if the core is generating the correct signals

Simple Adder Core using UART and Blockram

Verilog, Digital Design, Xilinx Vivado, Artix A7

Designed a simple core in verilog which gets two number input using UART interface in FPGA(the data to the uart is sent from the python code in the PC) and the data received store in Blockram in FPGA. Once both the numbers are stored in Blockram, we read it from the block ram and add the two numbers and store it the blockram and then send it back to the PC using UART interface
